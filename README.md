# Segmentasi Area Sawah dan Area Non Sawah

Segmentasi Area Sawah Web App adalah aplikasi web interaktif berbasis Streamlit yang dapat melakukan segmentasi area sawah dan non-sawah dari citra satelit.

Aplikasi ini menggunakan algoritma Split and Merge dengan fitur standard deviation map, GLCM (Gray Level Co-occurrence Matrix), dan warna LAB untuk menghasilkan segmentasi yang akurat.

Output dari aplikasi berupa citra satelit yang telah disegmentasi, diwarnai sesuai area sawah dan non-sawah, sehingga memudahkan analisis spasial dan pemetaan pertanian.
